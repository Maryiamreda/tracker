## Setup Instructions

### Prerequisites

- PostgreSQL database
- Supabase account
- Drizzle ORM

### Installation

1. Clone the repository

2. Install dependencies

```bash
npm install
# or
yarn
```

3. Start the development server

```bash
npm run dev
# or
yarn dev
```

4. Open your browser and navigate to `http://localhost:3000`
